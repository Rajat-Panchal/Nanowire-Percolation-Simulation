# Nanowire Percolation Simulation

This project investigates percolation phenomena using Python.

## Features

- Random nanowire network generation
- Source-drain connectivity analysis
- BFS-based path detection
- Percolation probability vs nanowire density
- Site percolation on 2D square lattices
- Finite-size effect analysis

## Tools Used

- Python
- NumPy
- Matplotlib
